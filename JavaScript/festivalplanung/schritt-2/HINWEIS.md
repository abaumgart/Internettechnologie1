# Schritt 2 – Line-up rendern (ca. 15 Min)

**Lernziel:** `forEach`, `createElement`, `className`, `innerHTML`, Funktion `lineupAnzeigen`.

## Wichtig: Styling nur über CSS
In JavaScript **kein** `kachel.style.border = ...`  
Stattdessen: `kachel.className = "kachel";`  
Das Aussehen steht in `styles.css` unter `.kachel { ... }`.

## Im Unterricht
1. Container leeren → neu aufbauen.
2. Klasse `kachel` zuweisen – Bezug zu vorbereitetem CSS aus Schritt 0.
3. Optional: in `styles.css` `.kachel` anpassen und Seite neu laden.

## Typische Fehler
- `getElementById` falsch geschrieben
- vergessen, Container zu leeren
- Styling versehentlich in JS statt in CSS
